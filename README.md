#  Mock E-Com Cart

## Overview
This is a **full-stack mock e-commerce shopping cart application** built for Vibe Commerce screening.  
It demonstrates basic e-commerce flows: product listing, add/remove from cart, total calculation, and mock checkout.  

**Tech Stack:**  
- **Frontend:** React.js, Axios  
- **Backend:** Node.js, Express.js, CORS  
- **Database:** Mock in-memory (JSON)  

## Features
- Display list of products  
- Add products to cart  
- Remove products from cart  
- Display total amount dynamically  
- Checkout form (mock receipt with total & timestamp)  
- Responsive design
